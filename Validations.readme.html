Welcome to PersistentLayer

Validations is a simple framework which contains a set of components responsable for validating any kind of objects through a set of rules named "Operations".

Latest Version

The quickest way to get the latest release of Validations is to add it to your project using NuGet (http://nuget.org/List/Packages/Validations).
